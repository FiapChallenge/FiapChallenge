JAVA CHALLENGE API

API para registro de acidentes para o site Guinchei

Endpoints:

  • GET - http://localhost:8080/JavaChallenge/api/caso
  • GET - http://localhost:8080/JavaChallenge/api/caso/query?nome=<nome>
  • GET - http://localhost:8080/JavaChallenge/api/caso/<id>
  • POST - http://localhost:8080/JavaChallenge/api/caso
  • PUT - http://localhost:8080/JavaChallenge/api/caso/<id>
  • DELETE - http://localhost:8080/JavaChallenge/api/caso/<id>
