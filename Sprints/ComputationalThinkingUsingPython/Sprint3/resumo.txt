📝| Descrição do projeto:
O projeto consiste em um programa em Python que permite a listagem, inserção, alteração e exclusão de modelos de veículos. 
O programa permite a inserção de dados como marca, modelo e tipo, e a partir disso, é possível realizar as operações citadas anteriormente.
Os dados são armazenados em um arquivo JSON, e o programa é executado no terminal.
O objetivo é visualizar o funcionamento de um CRUD (Create, Read, Update, Delete) em Python e a manipulação básica da base de dados.
